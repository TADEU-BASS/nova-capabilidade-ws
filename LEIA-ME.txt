CAPABILIDADE WS - POSTGRESQL - LOGIN POR TÉCNICO

LOGIN INICIAL:
Administrador:
  usuário: admin
  senha: 1234

Técnico inicial:
  usuário: marcos
  senha: 1234

IMPORTANTE:
- Troque as senhas após o primeiro acesso.
- Cada técnico vê somente os ensaios salvos pelo seu próprio usuário.
- Administrador vê todos os ensaios e pode cadastrar técnicos.
- Excluir ensaio exige perfil ADMIN e a senha APP_ADMIN_PASSWORD.
- No Render, use DATABASE_URL com a Internal Database URL do PostgreSQL.
- As antigas variáveis DB_HOST, DB_USER, DB_PASSWORD, DB_NAME e DB_PORT eram do MySQL e não são mais necessárias.

COMANDOS:
Instalar dependências:
  npm install

Iniciar localmente:
  npm start

Resetar admin localmente ou via ambiente configurado:
  npm run reset-admin
